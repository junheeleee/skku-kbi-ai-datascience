# 성균관대 KBI — AI 기반 데이터사이언스 학습 포트폴리오

> 성균관대학교 KBI 과정에서 이수한 **머신러닝**과 **딥러닝** 학습 내용을 정리한 포트폴리오입니다.  
> 이론 개념부터 PyTorch 기반 실습까지 데이터사이언스 전 스택을 다룹니다.

## 세부 학습 내용

### [머신러닝](./머신러닝/README.md)
- Python 기초 · NumPy · Pandas를 활용한 데이터 전처리
- Naive Bayes를 이용한 스팸 메일 분류 (Enron 데이터셋)
- SVM · 의사결정나무로 Iris 분류
- 선형 회귀로 키-몸무게 예측, 로지스틱 회귀로 Titanic 생존 예측
- K-Means 클러스터링, PCA · t-SNE 차원 축소, 이상치 탐지

### [딥러닝](./딥러닝/README.md)
- PyTorch Tensor 연산 및 Autograd 이해
- 다층 퍼셉트론(MLP)으로 MNIST 이진/다중 분류
- CNN으로 이미지 특징 추출 및 분류
- RNN · LSTM · GRU로 주가 · 교통량 시계열 예측
- BERT 파인튜닝 (한국어 NLP), GPT 텍스트 생성
- Diffusion Model을 이용한 이미지 생성, ChatGPT 활용 실습
